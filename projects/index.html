Featured Projects

Geovision: Image Geolocation with ViTs

Fine-tuned Vision Transformer (ViT-B/16) and ResNet-50 on a 104-class country geolocation task achieving 88.3% top-5 accuracy. Built an end-to-end pipeline with Hugging Face Transformers and Datasets APIs; performed error analysis, class balancing, and hyperparameter tuning.

RepWise

Designed and developed a React Native app generating personalized AI workout plans via Gemini API based on user history and goals. Integrated Supabase for backend, enabling secure workout history storage and retrieval with data validation.

SpotifyTracker

Built a React Native app using Spotify Web API to display users’ top tracks and artists across customizable time ranges.

Audio Feedback Research Project

Developed a voice dictation tool with named entity recognition and real-time error correction to enhance transcription accuracy for blind and low-vision users. Integrated Google Web Speech API for speech-to-text and Gemini API for proper noun detection; conducted a user study and authored a research paper on findings.

Multilingual Narrative Detection with XLM-R

Developed a multi-task XLM-R model for concurrent named entity recognition and sentiment analysis on multilingual social media political text. Implemented a two-phase fine-tuning approach and Bayesian hyperparameter optimization, improving performance across diverse linguistic datasets.
